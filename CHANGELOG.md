## [1.0.1] - 2026-03-02

### Fixed
- 修复日志查看模块分页功能（50 条/页）
- 修复日志刷新时消息内容不同步问题
- 修复堆栈跟踪识别功能（Java/Python 异常自动识别为 ERROR）
- 修复输入框无法输入问题（CSS 层级修复）

### Changed
- 优化日志解析器性能（预编译正则表达式）
- 改进 IPC 通信机制（增量读取避免重复推送）

## [1.0.0] - 2024-01-01

### Added
- 初始版本发布
- 支持 7+ 种常见日志格式自动解析
- 实时监控多个日志文件
- 日志级别过滤和关键字搜索
- 时间范围筛选功能
- 告警通知（桌面通知、声音告警）
- 仪表盘统计图表（ECharts）
- 监控任务配置管理
- 日志导出功能（CSV/JSON/TXT）
- 跨平台支持（Windows/Linux）

### Supported Log Formats
- 括号格式 `[2024-01-01 12:00:00] [ERROR] 消息`
- ISO 格式 `2024-01-01T12:00:00.000Z ERROR 消息`
- 简单格式 `2024-01-01 12:00:00 ERROR 消息`
- JSON 格式日志
- Apache 访问日志
- Syslog 格式

### Features
- 日志分页显示（50 条/页）
- 实时自动刷新（5 秒间隔）
- 堆栈跟踪自动识别
- 日志级别颜色标记
- 监控任务启动/停止控制
- 告警历史记录

## 版本说明

### 版本号规则
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型
- **Added**：新增功能
- **Changed**：现有功能的变更
- **Deprecated**：即将废弃的功能
- **Removed**：已删除的功能
- **Fixed**：错误修复
- **Security**：安全性修复
